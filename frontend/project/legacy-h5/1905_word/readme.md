## 微信分享工作记录

### 当前思路

* snsapi_base / 用户管理接口类: 无感知获取用户的openid, 需要用户关注公共号, 才可以成功回调, 获取用户信息
  * 跳转链接
  * 获取用户的openid
  * 通过openid和appid来获取`普通的access_token`
  * 判断是否过期
  * 使用`普通access_token`进行后面接口的调用
* snsapi_userinfo: 不需要用户关注公共号, 需要用户手动同意, 获取用户信息.
  * 获取到`网页授权access_token`, 通过token, 可以获取到用户信息
  * 跳转网页, 用户确认后, 跳回网页
  * 获取code, 后端处理, 换回用户信息
