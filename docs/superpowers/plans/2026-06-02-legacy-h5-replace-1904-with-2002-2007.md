# legacy-h5 Replace 1904 With 2002 And 2007 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove slow `1904_tale` from `legacy-h5` and add playable local-static versions of `2002_spell` and `2007_picky`.

**Architecture:** Keep one Vite multi-page project at `frontend/project/legacy-h5`. Each activity remains a subdirectory with its own HTML, JS, CSS/Less, and `asset/` tree. Backend, WeChat, tracking, sensitive-word, and save flows are replaced by deterministic local browser logic.

**Tech Stack:** Vite, plain JavaScript, Less, Hilo, jQuery/Zepto, Scroller, local no-op shims, local static assets.

    if (!normalized) {
      reject(new Error('输入为空'))
      return
    }

    }

}
